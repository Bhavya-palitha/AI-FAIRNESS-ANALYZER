# 🤖 AI Fairness Analyzer

## 📌 Project Overview
AI Fairness Analyzer is a web-based application designed to **detect bias in AI decision-making systems** and improve fairness.  
It helps ensure that automated decisions are **ethical, transparent, and unbiased** across different groups.

---

## 🎯 Problem Statement
Many AI systems (like hiring, banking, healthcare) show **bias towards certain groups** (e.g., gender, age).  
This leads to **unfair decisions and inequality**.

---

## 💡 Solution
This project:
- Detects bias in dataset decisions  
- Compares outcomes between groups  
- Applies simple fairness correction  
- Displays results clearly through UI and graphs  

---

## 🚀 Features
- ✅ Bias Detection (Male vs Female comparison)  
- ✅ Fairness Improvement (bias reduction)  
- ✅ Interactive Web Interface  
- ✅ Real-time Analysis Button  
- ✅ Visual Output (Before vs After Graph)  
- ✅ Simple & User-Friendly Design  

---

## 🛠️ Technologies Used
- Python  
- Flask (Web Framework)  
- Pandas (Data Processing)  
- Matplotlib (Visualization)  
- HTML + CSS (Frontend UI)  

---

## ⚙️ How It Works
1. User runs the web application  
2. Clicks **"Run Analysis"**  
3. System processes sample dataset  
4. Detects bias in decisions  
5. Applies fairness correction  
6. Displays:
   - Selection rates  
   - Bias result  
   - Graph visualization  

---

## 📊 Output
The system provides:
- Male vs Female selection rates  
- Bias detection result (Fair / Not Fair)  
- Graph showing improvement  
- Clear UI for understanding results
